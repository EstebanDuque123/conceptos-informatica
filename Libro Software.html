Conceptualización

¿Qué es Información?

Es un conjunto de datos con un significado en un determinado contexto.

¿Qué es dato?

Es el conjunto de caracteres (cifras,palabra,letra) que se almacena en la computadora en un determinado formato. Existen tres tipos básicos de datos:
-Numérico: 1, 23, 54...
-Alfabético: Formado por letras. Ej: a,x,pedro...
-Alfanuméricos: Formado por alfabeticos, numericos y especiales. Ej: Má6, valor4.

¿Qué es red?

Conjunto de ordenadores que se conectan entre sí e intercambian información constantemente. Las redes de computadoras se suelen clasificar según su alcance: Personales (PAN), locales (LAN), de campus (CAN), metropolitanas (MAN), amplias (WAN). O también pueden ser clasificadas según su conexión, es decir, cableada o inalámbricas (wireless).

Internet

Es la red de redes, es decir, una red que conecta computadores o máquinas a través de algún medio (cable coaxial, fibra óptica, líneas telefónicas,etc).

Navegadores

Los navegadores de Internet más conocidos son: Microsoft Internet Explorer (I.E), Mozilla, GoogleChrome.

¿Qué es un Bit?

Es un dígito de numeración binario. Esta representada por el uno (1) y el cero (0). Su unidad siguiente es el byte y es el conjunto de ocho bits ordenados.

¿Qué es un programa?

Es un conjunto de instrucciones que se usa para controlar las tareas de una máquina o computadora.

Los componentes o elementos básicos de un sistema informático son el hardware y el software.

Software

El software es el soporte lógico e inmaterial que permite que la computadora pueda desempeñar tareas inteligentes, dirigiendo a los componentes físicos o hardware con instrucciones y datos a través de diferentes tipos de programas.

Clasificación del Software:

El software se puede clasificar en tres grandes tipos: Software según uso, software según distribución y software según licenciamiento.

Software según uso

Encontramos tres tipos de software según uso: Software de sistema, Software de programación y Software de aplicación.

Software de sistema:

Conjunto de programas que sirven para interactuar con el sistema, confiriendo control sobre el hardware, además de dar soporte a otros programas. Ejemplo: Sistema operativo (Windows, Linux, entre otros), controladores de dispositivo.

Software de programación:

Conjunto de herramientas que permiten desarrollar programas informáticos, usando diferentes alternativas y lenguajes de programación de una manera práctica.


Software de aplicación

Permite a los usuarios llevar a cabo una o varias tareas específicas en la computadora. Ejemplo: software educativo, aplicaciones ofimáticas, videojuegos, etc.

Software según su distribución

El software se rige por licencias de utilización, es decir, en ningún momento un usuario compra un programa o se convierte en propietario así haya pagado por el, solo adquiere el derecho de uso. Este se compone de tres tipos de software: software de evaluación, software de demostración y software freeware.


Software de evaluación:

Se caracteriza porque es de libre distribución o copia, de tal forma que se puede usar contando con el permiso del autor durante un periodo ilimitado de tiempo.

Software de demostración:

Es similar al software de evaluación por la forma en la que se distrubye, pero en esencia es software propietario limitado que se distribuye con fines netamente comerciales.


Software Freeware:

Es software que se puede usar, copiar y distribuir libremente, pero que no incluye archivos fuente.

Software según su licenciamiento

Software propietario

Es software cerrado, donde el dueño del software controla su desarrollo y no divulga sus especificaciones. El software propietario es el producido principalmente por las grandes empresas, como Microsoft y otras más.

Software libre

Software que se puede usar para copiar, distribuir y modificar libremente.

Software de dominio publico

Es un software libre que tiene la ausencia de Copyright, es decir, es software libre sin derechos de autor.


Software semi-libre

Es software que posee la libertades del software libre, pero solo se puede usar para fines sin animo de lucro.

Sistema operativo

Un Sistema Operativo (SO) es la parte fundamental del software de cualquier computador, celular (smartphone), tablet, para que funcione correctamente; es decir, permite la comunicación entre el usuario y el ordenador.

Su función principal es darnos herramientas necesarias para poder controlar el dispositivo y poder hacer uso de ella de la forma más comoda y sencilla posible.

Ejemplo: Cuando se deseea imprimir algo. se entrega la tarea al SO. El SO entrega las instrucciones a la impresora utilizando los controladores de la impresora, y así, la aplicación que esta imprimiendo no tiene que entender que hace la impresora ya que el SO maneja todos los detalles.

Partes de un Sistema Operativo

El SO cuenta con muchas partes o componentes, pero lo tres componentes definidos y usados para todos los SO son tres: Kernel, interfaz de usuario, interaces de programación de aplicaciones.

Kernel:

Se puede decir que este es el corazón del sistema operativo y también es llamado como 'nucleo' del SO. Es el que proporciona un control básico sobre todos los dispositivos hardware del computador y es el primero en cargar cuando inicia el SO.

Interfaz de usuario:

Este componente permite la interacción con el usuario; puede darse por medio de graficos.

Interfaces de programación de aplicaciones

Este componente permite escribir código por partes a todos los desarrolladores.

Tipos de Sistema operativo

Existen sistemas operativos para computador y sistemas operativos para teléfonos móviles.

Sistemas operativos para computador

Los SO para pc sn muy variados, pero los más utilizados son: Windows, Linux y Mac.

Sistemas operativos para teléfonos móviles

Los SO más utilizados para estos dispositivos son el Android y iOS.

Antivirus

Los antivirus son programas que se crearon con el fin de buscar, identificar y detectar amenazas para después bloquearlas y eliminarlas del equipo.
